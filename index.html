Professional Self-Assessment

Completing the Computer Science program has helped me sharpen my technical, collaborative, and problem-solving abilities. The ePortfolio serves as a comprehensive reflection of this journey, allowing me to demonstrate the progression of my skills through selected and enhanced artifacts. Throughout the program, I developed a strong foundation in designing scalable software systems, working with both front-end and back-end technologies. One of my most impactful projects was Travlr Getaways, a full-stack travel management application built with Angular, Node.js, and MongoDB. This project evolved over time and became the foundation for demonstrating all three enhancement categories: software design and engineering, algorithms and data structures, and databases. The enhancements I implemented for the capstone included integrating Angular Material components for a professional, accessible UI, improving user input validation logic, and introducing soft-delete functionality to preserve data integrity. These changes elevated the usability, maintainability, and professionalism of the application. For algorithms and data structures, I chose a checkers game AI from a prior project. The original logic was functional but limited in strategic depth. I analyzed the decision-making flow and implemented a more robust board evaluation algorithm, enhancing the AI’s competitiveness and showcasing my ability to critically assess and optimize logic using algorithmic principles. My understanding of database systems matured significantly through building CRUD functionality, implementing soft-deletes in MongoDB, and ensuring strong data consistency. By simulating real-world needs for reversible deletions, I learned how to enhance data integrity and user trust in persistent storage systems. Security and secure coding practices were embedded throughout my capstone enhancements. I paid attention to input sanitization, database validation, and protecting backend routes – practices that are essential in today’s security-focused landscape. Another key area of growth was team collaboration and communication. By using Git and GitHub throughout the program and especially in the capstone, I became confident in version control workflows, resolving merge conflicts, and providing readable documentation. The ability to translate complex logic into simple explanations and collaborate with peers will be invaluable in a professional setting. This ePortfolio reflects my specialization in full-stack development and my aspiration to work in the music tech industry with platforms like Spotify or Apple Music. I am passionate about building tools that enhance creativity, and my skills in web development, algorithms, and system design are aligned with that vision. The following sections contain three artifact enhancement narratives organized by category: Software Design & Engineering, Algorithms & Data Structures, and Databases. Each narrative explains the original project, the enhancements applied, the challenges faced, and the course outcomes achieved. Together, they form a cohesive story of how SNHU's Computer Science program prepared me for a successful career in software development.

Databases Enhancement

Artifact: Travlr Getaways

Original Project: Developed in CS-465 Full-Stack Development

Enhancement Summary: I implemented soft-delete functionality for trip records by adding a deleted flag instead of permanently removing documents. This is a critical industry feature to prevent data loss and enable restoration of accidentally removed records.

Skills Demonstrated:

  • Advanced MongoDB query operations
  • Data persistence best practices
  • Simulating production-like data handling patterns

Learning Reflection: Implementing soft-deletes forced me to rethink how data is queried and displayed. I had to modify all find() and update() queries to ignore soft-deleted items while maintaining backend logic that could restore them. This enhancement aligns with real-world scenarios and improved the overall robustness of the application.

Course Outcomes Met:

  • Develop software systems that integrate security best practices at every stage of the development lifecycle.
